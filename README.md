# Prueba individual de proyecto integrador de software


Asegurate primero de que el archivo `output.json` dentro del directorio backend/database/seed/
# Pasos de ejecucion
# iniciar la base de datos

```bash
  docker-compose up -d
```
# instalar dependencias
```bash
  cd backend
  npm install

```


# ejecutar la aplicacion
```bash
  npm run start:dev
```
# Acceder a la documentacion de la API
```bash
  http://localhost:3000/api
```
